# Library Management System (Go)

## Overview
A simple console-based library management system demonstrating structs, interfaces, slices, maps, and methods in Go.

## Features
- Add / Remove books
- Borrow / Return books
- List available and borrowed books

## Components
- **models/**: Defines data structures (Book, Member)
- **services/**: Contains core logic and implements LibraryManager interface
- **controllers/**: Handles user input and console interaction
- **main.go**: Entry point
